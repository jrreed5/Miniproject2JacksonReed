# Airplane Crash Data Visualizer

## Description

This program uses a CSV from Kaggle to visually represent what 20 air operators have the most fatalities in total. 

## Getting Started

### Dependencies
* Python 3.12.x
* Pandas (2.2.0) 
* matplotlib (3.8.2)
* pathlib (1.0.1)

### Installing

* clone this repository

### Executing program

* Run ./main.py
